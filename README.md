# Sandplotter---DA
Diese App ist eine Steuerungsapp für unsere Diplomarbeit - den Sandplotter

Um die App zu installieren, laden Sie die APK-Datei herunter und installieren Sie diese auf ihrem Anroid-Gerät.
Um die Bluetooth-Verbindung zu dem Sandplotter zu ermöglichen, aktivieren Sie bitte den Standort auf Ihrem Gerät.
